타자왕 모바일 가상 키보드 차단 패치

수정 파일
- king.html
- assets/js/king.js

적용 내용
1. 모바일 타자왕 입력창에 inputmode=none 적용
2. 터치/포커스 시 readonly 포커스 방식으로 휴대폰 가상 키보드 호출 방지
3. 외장 키보드/속기 키보드 입력은 유지
4. 문장 전환 시 기존 스크롤 위치 보존 로직 유지
5. king.html의 king.js 캐시 버전값 변경

주의
- 휴대폰 화면 자판을 이용한 입력은 의도적으로 막힙니다.
- 타자왕은 외장 키보드/속기 키보드 연결 사용 기준입니다.
