AI 선생님 피드백방 UI 고도화 수정본

목적
- 기능을 계속 추가해도 화면이 복잡해지지 않도록, 추천 기능을 별도 카드로 길게 나열하지 않고 'AI 코칭 보드' 한 섹션에 압축 배치했습니다.

반영 내용
1. AI 코칭 보드 추가
   - AI 진단 점수
   - 승급 가능성
   - 약어 숙련도
   - AI 이용권

2. 이번 주 학습 처방전 추가
   - 최근 기록, 정확도, 오류 유형을 기준으로 이번 주 우선 학습 방향을 자동 표시합니다.
   - 오자/탈자/첨자/띄어쓰기 중 가장 우선 점검할 항목을 처방 문장으로 보여줍니다.

3. 추천 액션 추가
   - 오늘 전체 분석 생성
   - 내 약점 전용 연습
   - 최근 7일 성장 분석
   - 선생님 확인용 요약

4. UI 정리 원칙
   - 기존 학습 리포트와 분석 보관함 사이에 배치했습니다.
   - 기능을 모두 펼치지 않고 점수판 + 처방전 + 액션 3단 구조로 압축했습니다.
   - 기존 AI 피드백 API, 학습 기록, 보관함 로직은 건드리지 않았습니다.
