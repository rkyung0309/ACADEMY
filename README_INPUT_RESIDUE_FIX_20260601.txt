타자왕/연습 화면 이전 입력 잔상 방지 패치 (2026-06-01)

수정 목적
- 모바일/한글 IME/외장 키보드 환경에서 다음 문장 또는 다음 문제로 넘어갈 때 이전 문장의 끝부분이 입력창에 늦게 남아 오답 처리되는 현상 방지

수정 파일
- assets/js/king.js
- assets/js/sparta.js
- assets/js/random.js
- assets/js/exam.js
- assets/js/intermediate.js
- assets/js/unlock-room.js

핵심 적용 사항
1. 타자왕
   - 문장 완료 시 기존 input의 input 이벤트 제거
   - 기존 input value/defaultValue 완전 초기화
   - 기존 input blur/readOnly/disabled 처리 후 제거
   - 새 input 생성 직후 이전 문장 끝부분이 밀려 들어오면 자동 제거

2. 스파르타/입문/약어랜덤
   - 다음 줄/다음 문제/오답 리셋 시 입력값 완전 초기화
   - 이전 문제의 끝부분이 조합 입력으로 늦게 들어오면 자동 제거

3. 중급/고급/급수 공부방 및 해금방
   - 연습 초기화/과목 전환 시 textarea 입력 잔상 방지
   - 이전 입력 일부가 늦게 들어와 글자수 카운트가 튀는 현상 방지

적용 방법
- 압축을 풀어 기존 프로젝트에 같은 경로로 덮어쓰기
- 배포 후 브라우저 캐시를 비우거나 새로고침 강제 실행(Ctrl+F5)

테스트 권장
- 타자왕에서 한 문장을 끝까지 입력한 뒤 다음 문장 입력창이 빈칸으로 시작되는지 확인
- 다음 문장으로 넘어간 직후 이전 문장 끝글자가 남지 않는지 확인
- 입문반/약어랜덤/스파르타에서 정답 후 다음 문제 입력창이 빈칸인지 확인
