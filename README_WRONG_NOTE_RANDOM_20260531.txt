오답노트 간소화 수정본

적용 파일
- assets/js/study-tools.js
- assets/css/study-tools.css
- assets/js/toksori2.js
- assets/js/editorial.js

수정 방향
1. 오답노트 화면을 복잡한 대시보드형 구조에서 오답 랜덤 연습방으로 단순화했습니다.
2. 채점 기록의 원문/입력문 스냅샷을 비교해 실제 오자·탈자·첨자 조각을 자동 추출합니다.
3. 약어랜덤처럼 Start → 랜덤 출제 → 입력 후 Enter → 바로 채점 → 다음 문제 흐름으로 동작합니다.
4. 정답 처리된 오답 조각은 완료 처리되어 다음 오답 랜덤 목록에서 제외됩니다.
5. 똑소리2와 사설연습도 앞으로 원문/입력문 스냅샷을 저장하도록 보강했습니다.

주의
- 기존 기록 중 원문/입력문 스냅샷이 없는 기록은 실제 오답 조각을 만들 수 없습니다.
- 중급반 공부방 기록은 이미 source_text_snapshot/typed_text_snapshot 저장 구조가 있으므로 새 채점 기록부터 바로 반영됩니다.
