똑소리 UI 전용 개편본

수정 파일
- toksori2.html
- assets/css/toksori2.css

수정 기준
- assets/js/toksori2.js는 수정하지 않았습니다.
- 기존 id와 onclick 함수 호출은 유지했습니다.
- 채점 로직, 저장 로직, 차트 로직, 오답노트 로직은 그대로 사용합니다.

개편 내용
- 첫 화면을 원문 준비 → 답안 입력 → 채점 확인 순서로 재배치했습니다.
- 실시간 연습왕 TOP 3은 하단 접힘 영역으로 이동했습니다.
- 최근 기록은 접힘 영역으로 이동했습니다.
- 번문 비교 채점은 고급 옵션으로 숨겼습니다.
- 결과 화면은 정확도/그래프/오답노트를 먼저 보이고, 상세 비교는 펼쳐보기로 정리했습니다.
- 모바일에서 입력 영역과 버튼이 한 줄로 깨지지 않도록 반응형 스타일을 추가했습니다.
