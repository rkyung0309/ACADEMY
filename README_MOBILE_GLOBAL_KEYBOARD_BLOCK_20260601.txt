소리자바 아카데미 모바일 가상키보드 차단 패치 - 2026-06-01

적용 목적
- 로그인/회원가입 같은 계정 입력 화면을 제외하고 모바일 학습 화면에서 휴대폰 가상키보드가 올라오지 않도록 처리했습니다.
- 외장 키보드/속기 키보드 입력은 유지합니다.
- JS로 나중에 생성되는 입력창도 자동 감지하여 inputmode="none"을 적용합니다.

주요 적용 화면
- 입문반 졸업시험
- 중급반/고급반/급수반 공부방
- 사설연습
- 약어랜덤
- 타자왕
- 오답노트
- 똑소리2
- 자수측정
- 스파르타
- 해금방/승급실
- 관리자/AI/마이페이지 등 일반 입력 영역

수정/추가 파일
- assets/js/mobile-keyboard-blocker.js 신규 추가
- 각 주요 HTML에 mobile-keyboard-blocker.js 연결
- assets/js/king.js 포함: 기존 타자왕 모바일 스크롤/자판 차단 수정 유지
- assets/css/mobile-responsive.css 포함: 기존 모바일 레이아웃 수정 유지

예외 처리
- index.html/editorial.html 내부 #login-wrapper 로그인 입력칸은 가상키보드 허용
- signup.html은 계정 생성 화면이므로 이번 차단 스크립트 연결에서 제외

적용 방법
- 압축 해제 후 기존 프로젝트 루트에 그대로 덮어쓰기
- 배포 후 모바일 브라우저 캐시가 남아 있으면 새로고침 또는 캐시 삭제 후 확인
