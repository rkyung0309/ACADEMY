# 오답노트 실제 오답 구간 복습 개편 (2026-05-31)

## 변경 요약
- 오답노트를 기존 '취약 파일 요약' 중심에서 '실제 틀린 구간' 중심으로 개편했습니다.
- practice_records의 source_text_snapshot / typed_text_snapshot을 비교해 오자·탈자·첨자 구간을 자동 추출합니다.
- 오답노트 화면에서 각 오답 구간의 내 입력, 정답, 문맥을 바로 확인할 수 있습니다.
- 오답 구간을 선택하면 우측 연습 패널에서 정답 문맥을 다시 입력하고 즉시 채점할 수 있습니다.
- 같은 오답 구간을 2회 정답 처리하면 자동 완료로 이동합니다.
- 수동 '완료 처리'도 가능합니다.
- 기존 '오늘 복습할 파일', '오답 원인 선택', '약어 오답 자동 복습함', '재응시 비교' 기능은 유지했습니다.

## 저장 보강
- 똑소리2 저장 시 원문/입력문 스냅샷과 오답 요약을 함께 저장하도록 보강했습니다.
- 사설연습 저장 시 원문/입력문 스냅샷과 오답 요약을 함께 저장하도록 보강했습니다.
- 중급반 공부방은 기존에 이미 원문/입력문 스냅샷과 diff_summary를 저장하고 있어 오답노트에서 바로 활용합니다.

## 확인 방법
1. 똑소리2 또는 사설연습 또는 중급반 공부방에서 일부러 오답을 낸 뒤 채점합니다.
2. 오답노트로 이동합니다.
3. '실제 오답 구간'에 오자·탈자·첨자 카드가 뜨는지 확인합니다.
4. '이 구간 연습'을 눌러 우측 연습 패널로 이동합니다.
5. 정답 문맥을 입력하고 '연습 채점'을 누릅니다.
6. 2회 정답 처리 후 해당 구간이 완료 목록으로 이동하는지 확인합니다.
