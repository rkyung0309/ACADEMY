AI 선생님 분석 대시보드 전면 노출 수정본

적용 파일:
- index.html
- mypage.html
- assets/js/learning-portal.js
- assets/css/ai-teacher-dashboard.css

반영 내용:
1. 대시보드 상단에 AI 선생님 분석 프리미엄 상품 영역 추가
2. 사이드바 홈 메뉴에 AI 선생님 분석 바로가기 추가
3. 대시보드에서 바로 오늘 전체 AI 피드백 생성 가능
4. 대시보드에서 바로 최근 7일 성장 분석 생성 가능
5. 오늘 기록 수, 최근 7일 기록 수, AI 분석 수를 전면 표시
6. 마이페이지 상단 버튼도 AI 선생님 분석으로 바로 이동하도록 조정

주의:
- 기존 AI 피드백 API, SQL, 입력 로그, 약어 분석 로직은 그대로 유지합니다.
- 이번 수정은 상품 노출/전환율 개선 목적의 UI 배치 변경입니다.
- Supabase SQL은 다시 실행할 필요 없습니다.
